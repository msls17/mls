# mls
莫老师のjio本

wx开头的脚本不用抓包，使用微信code登录，需搭建WeChatPadPro
项目地址https://github.com/WeChatPadPro/WeChatPadPro
搭建并登录好微信后，设置变量名serviceip，值为WeChatPadPro服务的ip和端口，如192.168.1.110:1238
变量名adminkey，值为WeChatPadPro的adminkey



需要安装以下linux依赖
uuidgen
jq
curl
grep
nss
openssl

以及以下python依赖
cryptography
pycryptodome
pycryptodomex
